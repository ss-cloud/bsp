---

支持shadowsocks-python和shadowsocks-Go

 **安装** 

```
git clone https://github.com/edboffical/bsp.git
cd bsp
chmod 775 install
./install
```

详细使用请查阅使用文档 [wiki](https://github.com/edboffical/bsp/wiki/bsp%E7%9A%84%E5%AE%89%E8%A3%85%E4%BD%BF%E7%94%A8)
